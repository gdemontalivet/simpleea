# Looker Visualization Types

This document lists the valid `type` values for the `vis_config` object. You must use one of these types when generating a visualization.

## Valid Visualization Types

- `looker_column`
- `looker_bar`
- `looker_line`
- `looker_pie`
- `looker_area`
- `looker_scatter`
- `single_value`
- `looker_grid`
- `table`
- `looker_map`
